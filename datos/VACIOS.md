- **Para actualizar estados:** edita el campo `estado` en el archivo `datos/hopelchen/HOPELCHEN_PREGUNTAS_<nodo>.json` correspondiente, luego ejecuta `python tools/actualizar_vacios.py`.
- **Al encontrar una respuesta:** cambia el estado a `RESPONDIDA PARCIALMENTE` o `RESPONDIDA` y agrega el campo `registro_respuesta` con el ID del registro JSON donde se guardó la evidencia.
- **Al identificar un nuevo vacío:** agrega la pregunta al archivo PREGUNTAS correspondiente con un `pregunta_id` nuevo; luego ejecuta este script.
